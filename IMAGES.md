# Inventaire des images bikeverbier.ch

Toutes les images sont servies depuis le CDN Framer (`framerusercontent.com/images/<id>`).
**Reco** : récupérer les **originaux HD** dans tes archives design (sources Tiffany Troillet / tes exports) plutôt que la version compressée Framer. L'URL Framer sans paramètres `?width&height` donne l'original, mais c'est souvent déjà ré-encodé.

Format à produire pour le web : WebP + fallback JPG, plusieurs tailles (`srcset`), lazy-loading sauf le hero.
